# dwm 6.1 (customized)
This is a dynamic window manager from suckless.org, with the following patches[^1] applied:
* autoresize
* bottomstack
* center
* [fancybarclickable](https://github.com/nggit/dwm-fancybarclickable-6.1)
* gaplessgrid
* pertag
* savefloats
* sticky
* systray
[^1]: Some patches have been modified / fixed.
