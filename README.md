# dwm 6.1 (customized)
This is a dynamic window manager from suckless.org, with the following patches<sup id="a1">[[1]](#f1)</sup> applied:
* autoresize
* bottomstack
* center
* dualstatus
* [fancybarclickable](https://github.com/nggit/dwm-fancybarclickable-6.1)
* gaplessgrid
* pertag
* savefloats
* sticky
* systray
